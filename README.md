# Project Cruise

Project Cruise is a cruise line for magical creatures — hidden from the human world — built to do two things at once: encourage tourism between realms and serve as **neutral diplomatic ground** where even rival peoples can travel, relax, and actually talk. Guests are verified before boarding, and every species is accommodated to its nature.

Built as a Scrum group project for CIS.
